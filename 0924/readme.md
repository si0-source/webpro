CSS3 스타일 시트는 셀렉터, 프로퍼티, 값으로 구성  
스타일을 적용하는 세 가지 방법 인라인 스타일, <style> 태그, 외부 스타일  
부모 태그로부터 상속  
이 4가지는 전부 쓰인다 content padding border margin  
border-radius나 shadow도 가능
display로 출력되는 방식 수정가능  
position으로 위치를 정할 수 있다  
마우스가 올라오면 hover  
포커스 마우스 클릭시 focus  
자바 스크립트로는 힘든 것을 css3로는 가능  
애니메이션(animation) 연속적인 움직임  
전환(transition) 서서히 변화시키기  
변환(transform) 회전 기울임 이동 확대/축소
